# ohnkyta-bot

QQ 聊天机器人 练习用

## 部署

- 请全局安装`bunyan`,`cross-env`，否则你需要手动指定启动方式

- 依赖于 HTTP API 插件

## 功能

- 随机符卡
  爬取解析 THBWiki 的符卡列表
  使用例
  `随机符卡`
- 随机色图
  色图来源：api.lolicon.app/setu/
  使用例
  `/色图`
- 直播提醒
  需要号主权限
  使用例
  `直播订阅 [房间号] [显示的主播名称]`
- MC 服务器打雷提醒
  需要配套打雷提醒 mod
  验证仍未完成
